# Modelare-si-Simulare-a-Productiei-de-Energie-prin-Panouri-Fotovoltaice
Scopul acestui proiect este modelarea matematica si simularea in MATLAB/Simulink a unui panou fotovoltaic, pentru a analiza productia de energie în functie de parametri precum: iradierea solara, temperatura, unghiul de incidenta și sarcina conectata
